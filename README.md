# 번개장터 플러스

번개장터 웹 버전 사용을 좀 더 편하게 만들어주는 Chrome 확장 프로그램입니다. 

상품 리스트에서 상품의 배송비를 바로 확인할 수 있고, 검색 결과 내 상품 리스트 사이사이에 존재하는 외부 광고 상품을 블라인드 처리해줍니다.

## 주요 기능

### 상품 리스트에서 배송비 표시

상품 리스트에서 상세 페이지를 들어가지 않고도 상품의 배송 옵션 및 배송비를 확인할 수 있습니다.
- **배송비 표시 방식 설정**: 사용자의 설정에 따라 배송비만 표시하거나 또는 배송비 포함 가격을 선택하여 표시할 수 있습니다.

### 광고 상품 블라인드

쇼핑 리스트 사이사이에 배치된 파워링크 및 외부 광고 상품을 반투명하게 블라인드 처리합니다.
- **광고 차단 설정**: 사용자의 설정에 따라 광고 상품을 블라인드 처리합니다.

## 작동 방식

상세 페이지에서 제공하는 배송비 정보를 상품 리스트에서 하나씩 가져와서 보여줍니다.

- **지연 로딩**: 모든 상품 정보를 한꺼번에 불러오는 대신, 화면에 노출되는 시점에 맞추어 데이터를 요청합니다. 이를 통해 브라우저 리소스와 네트워크 트래픽을 효율적으로 관리합니다.
- **요청 대기열**: 짧은 시간 내에 과도한 요청이 발생하지 않도록 대기열 시스템을 운영합니다. 정해진 간격에 따라 순차적으로 데이터를 처리하여 안정성을 확보합니다.
- **배송비 정보 캐싱**: 이미 한 번 조회된 상품 정보는 브라우저 내부에 안전하게 저장됩니다. 동일한 상품을 다시 볼 때는 서버 통신 없이 즉시 가격 정보를 표시합니다.

---

이 프로젝트는 개인적인 편의를 위해 제작되었으며, 번개장터 공식 서비스와 무관합니다.
