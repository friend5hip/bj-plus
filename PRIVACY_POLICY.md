# 번개장터 플러스 개인정보처리방침

**번개장터 플러스**(이하 '확장 프로그램')는 사용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수하기 위해 다음과 같은 개인정보처리방침을 수립하여 공개합니다.

## 1. 수립 및 목적
본 확장 프로그램은 번개장터 웹사이트 이용 시 배송비 정보를 통합하여 표시하고, 광고 요소를 차단하여 사용자 편의를 증진하는 것을 목적으로 합니다.

## 2. 수집하는 데이터 및 사용 목적
본 확장 프로그램은 사용자의 개인 식별 정보를 **절대 수집하거나 외부 서버로 전송하지 않습니다.** 다만, 서비스 제공을 위해 아래와 같은 데이터를 브라우저의 로컬 환경(Local Storage)에 일시적으로 저장합니다.

*   **설정 데이터**: 사용자가 선택한 배송비 표시 방식 및 광고 차단 활성화 여부를 저장하여 설정 상태를 유지합니다.
*   **캐시 데이터**: 동일한 상품에 대한 중복 API 호출을 방지하기 위해, 조회된 상품의 ID와 배송비 정보를 브라우저 내에 임시 저장(캐싱)합니다.

## 3. 데이터의 공유 및 제3자 제공
본 확장 프로그램은 수집되거나 로컬에 저장된 어떠한 데이터도 제3자에게 판매, 대여 또는 공유하지 않습니다. 모든 데이터 처리는 사용자의 브라우저 내부에서만 이루어집니다.

## 4. 데이터의 보유 및 파기
*   사용자는 확장 프로그램을 삭제함으로써 저장된 데이터를 즉시 파기할 수 있습니다.
*   저장된 데이터는 브라우저의 로컬 스토리지에 귀속되며, 확장 프로그램 제거 시 함께 삭제됩니다.

## 5. 보안
본 확장 프로그램은 사용자 데이터를 외부 서버로 전송하지 않습니다. 기능 실행을 위해 번개장터 공식 API와 통신하지만, 이 과정에서 어떠한 사용자 식별 정보나 민감한 정보를 포함하지 않습니다.
